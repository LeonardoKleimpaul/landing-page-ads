# Landing Page da Floricultura da Ana

## Sobre o Projeto

Ana, empreendedora e apaixonada por flores, solicitou a criação de uma página inicial estática para destacar sua floricultura. Com o objetivo de proporcionar uma experiência visual envolvente aos seus clientes online, o desafio foi desenvolver uma página web utilizando tecnologias essenciais para web, como HTML e CSS.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Estrutura do Projeto

O projeto está organizado da seguinte forma:

- **`index.html`**: Arquivo principal da landing page.
- **`style.css`**: Arquivo de estilização da página.
- **`main.js`**: JavaScript para funcionalidades interativas.
- **`images/`**: Diretório contendo imagens utilizadas na página.
